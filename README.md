# Phishing Awareness Training

This project is a phishing awareness training presentation created for the CodeAlpha Cyber Security Internship. The goal of this training is to help users understand how phishing attacks work and how to avoid falling victim to them.

## What this project covers

* Introduction to phishing attacks
* How to recognize phishing emails
* How to identify fake websites
* Common social engineering tactics
* Best practices to avoid phishing attacks
* Real-world phishing examples
* Interactive quiz for better understanding

## Purpose

Phishing is one of the most common cyber threats. This training module is designed to spread awareness and help users identify suspicious emails, links, and websites before sharing sensitive information.
